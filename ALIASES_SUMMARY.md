## How It Works

With the gozzi static site generator's alias feature:
1. User visits `/blog/neovim-debug`
2. Gozzi generates HTML redirect at that path
3. Redirect uses `<meta http-equiv="refresh">` to canonical URL
4. Also includes `<link rel="canonical">` for SEO
5. Fallback link in body for accessibility

## Benefits

### SEO Improvements
- Multiple URL paths to same content
- Keyword-rich URLs rank better
- Canonical links prevent duplicate content issues

### User Experience
- Shorter URLs easier to remember and share
- Topic-based URLs (`/blog/neovim-*`) are intuitive
- Date-less URLs feel timeless

### Link Preservation
- Old links keep working
- Can restructure content without breaking links
- Multiple entry points for discovery
